void tacho ( )

Display the current speed on tacho_row LEDs.

It is show from left to right in three different color.

  • Green (0x001100): low speed
  • Orange (0x110800): medium speed
  • Red (0x110000): ludicrous speed

The function getspeed() returns an unsigned integer with a step size of 1 equals 0.4302 km/h. As the speed is displayed in three color the maximum speed should be dividable by three.

The integer value 30 -> 12.9 km/h. 12.9 km/h should be enough for small children. If you have a older child or use this for different purposes (e.g. a bicycle), the maximum speed should be higher.

30/3 = 10 -> speed * count of tachometer LEDs / 10
